# Практика 2
## О проекте
Базовая форма сайта по предоставленному макету.
Для ситлей CSS использзована методология БЭМ в связи с темЮ, что он идеально подходит для создания независимых компонентов (например, .card или .button), которые можно повторно использовать в разных частях страницы без конфликтов стилей

## Используемые технологии
- HTML5
